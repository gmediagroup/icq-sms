Установка.

1. Скопируйте все папки и файлы на сервер;

2. http://ваш_домен/admin/ - это админка

по умолчанию заданы:
логин: admin
пароль: admin

3. Введите свой SMS API ID;

4. Обновите (.CSV) базу номеров на странице "SMS Numbers Groups"; 

5. Создайте группы номеров по инструкции на странице "SMS Numbers Groups";
	Имя группы лучше называйте в виде цены, например "30 руб", так будет Вам самим понятнее.

6. В разделе "ICQ UIN base" добавьте товары в базу данных.

Enjoy!
